ملف الموقع 'كونكت' — تعليمات سريعة
-----------------------------
الملفات المضمنة:
- index.html   : الصفحة الرئيسية (RTL عربى)
- styles.css   : تصميم الصفحة
- script.js    : منطق حفظ البيانات في localStorage، وإضافة/بحث/حذف
- README.txt   : هذا الملف

كيفية التشغيل (محلياً):
1. فك الضغط عن الملف connect_website.zip.
2. افتح المجلد، ثم افتح index.html في المتصفح (يفضل Chrome أو Edge).
3. الموقع يعمل دون خادم؛ البيانات محفوظة في متصفحك (localStorage).
4. إن أردت ربطه بخادم/قاعدة بيانات لاحقاً سأقوم بمساعدتك.

ملاحظات فنية:
- الصور تُخزن بصيغة Data URL داخل localStorage. الصور الكبيرة قد تزيد من حجم التخزين.
- إذا أردت خاصية الحفظ في قاعدة بيانات (PHP/MySQL أو Node.js) أخبرني وسأرفق الملفات اللازمة.
